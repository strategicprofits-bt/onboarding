Performance Golf AI Onboarding

This onboarding configures Claude as your AI working partner, calibrated to your role and your workflows. The full process takes approximately 60 minutes across four steps.

This onboarding is powered by Strategic Profits, Performance Golf’s AI implementation partner.

1 Book Your Call 2 Set Up Claude 3 AI Mapping 4 Client Profile
1

Book Your Call with Ben

~5 min

Step 1 of 4

Your first step is booking a one-on-one call with Ben — a pre-training check-in session to make sure you are ready to hit the ground running. No preparation is required beyond completing Steps 2–4.

Schedule your call at least 3–5 days out. Ideally complete Steps 2–4 before the call, but if you get stuck on something, you can work through that on the call.

Book your call at meetben.com/pg →

✓
You're done when: You have a confirmed call on the calendar.
2

Set Up Claude

~10 min

Step 2 of 4

You will need Claude Code installed and connected to the Strategic Profits Shared MCP server. Choose whichever installation method you prefer. Steps 2B and 2C require basic comfort with a terminal. Let us know in the Slack channel if you need assistance.

Note: you may have already completed these steps as part of earlier work with Claude Code.

Option A: Desktop App

Download and install the Claude desktop app from claude.ai/download. Open it and sign in with your Anthropic account.

Option B: CLI (Terminal)

If you prefer working in a terminal, install Claude Code via npm:

npm install -g @anthropic-ai/claude-code

Then run claude in your terminal and follow the sign-in prompts.

Connect Strategic Profits Shared MCP

Strategic Profits Shared is the platform that gives your Claude access to your program’s skills, interview tools, and submission system. Connecting it lets Claude run your AI mapping, client profile, and automatically deliver your results to the team.

You'll need: your Strategic Profits API key (starts with sk_sp_, emailed to you when you enrolled).

Need your API key? Enter your work email below and we will send it to you. If this is your first time here, your API key will be sent to your work email after you submit the form.

Once Claude Code is running, paste the following into the Claude chat box (not your system terminal), replacing YOUR_SP_API_KEY_HERE with your API key:

RUN THIS COMMAND Add this MCP server to my project config: name "sp-shared", URL "https://mcp.strategicprofits.com/mcp", with header "Authorization: Bearer YOUR_SP_API_KEY_HERE". Then run sp_sync in full mode to download shared skills.
✓
You're done when: Claude confirms sp_sync completed and skills were downloaded.
3

Complete Your AI Mapping

~20–30 min

Step 3 of 4

The AI opportunity interview maps your team’s operations, workflows, and pain points to identify where AI can deliver the most value across your organization.

Open Claude Code and paste the following into the chat box — Claude will handle the rest:

RUN THIS COMMAND Run the AI opportunity interview. Use the ai-opportunity-interview skill. When the interview is complete, submit the results using the sp_submit_ai_opportunity tool.

This is a 20–30 minute structured conversation — not a form, a back-and-forth interview. Be specific and candid; there are no wrong answers. If you need to step away, progress is saved automatically.

When the interview is complete, Claude automatically submits your results to the team.

✓
You're done when: Claude confirms your AI Opportunity Profile and Command Center Brief were submitted.
4

Client Profile

~20–30 min

Step 4 of 4

The AI Mapping interview was about how your team works. This interview is about how you work — it’s what Claude uses to configure itself specifically for you.

This is a deeper interview covering your responsibilities, tools, pain points, goals, and how you prefer to work with AI.

Open Claude Code and paste the following into the chat box — Claude will handle the rest:

RUN THIS COMMAND Run the client profile interview. Use the client-profile-interview skill. Save my profile to CLIENT-PROFILE.md and save progress as we go. When complete, submit it using sp_submit_onboarding with program "pg".

Be thorough — Claude uses your answers to build your entire AI configuration. If you need to step away, progress is saved automatically.

When the interview is complete, Claude saves your profile locally and submits it to the team.

✓
You're done when: Claude confirms your client profile was submitted.
