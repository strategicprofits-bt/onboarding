Three steps to get up and running. Start at the top and work your way down.

1 Book Your Call 2 Set Up Claude 3 AI Mapping 4 Grounding Package
1

Book Your Call with Ben

~5 min

Step 1 of 4

Your first step is a one-on-one call with Ben to map out your AI strategy and make sure you're set up for success.

Book your call at meetben.com/pg →

✓
You're done when: You have a confirmed call on the calendar.
2

Set Up Claude

~10 min

Step 2 of 4

You'll need Claude Code installed and connected to the SP Shared MCP server. If you've already done this, skip to Step 3.

Install Claude Code

If you don't have Claude Code yet, install it from claude.ai/download (Desktop app) or run npm install -g @anthropic-ai/claude-code in your terminal (CLI).

Connect SP Shared MCP

Open Claude Code and paste this command, replacing YOUR_SP_API_KEY_HERE with your API key:

RUN THIS COMMAND Add this MCP server to my project config: name "sp-shared", URL "https://mcp.strategicprofits.com/mcp", with header "Authorization: Bearer YOUR_SP_API_KEY_HERE". Then run sp_sync in full mode to download shared skills.
✓
You're done when: Claude confirms sp_sync completed and skills were downloaded.
3

Complete Your AI Mapping

~20–30 min

Step 3 of 4

Before your call with Ben, complete the AI opportunity interview. This maps out your business, workflows, pain points, and where AI can make the biggest impact for you.

Open Claude Code and run:

RUN THIS COMMAND Run the AI opportunity interview. Use the ai-opportunity-interview skill.

Claude will have a 20–30 minute conversation with you — not a form, an actual back-and-forth interview. Answer honestly and specifically. There are no wrong answers. If you need to step away, progress is saved automatically.

When You're Done

The interview produces two files:

  • AI-OPPORTUNITY-PROFILE-[YourName]-[Date].md — your full AI opportunity profile
  • COMMAND-CENTER-BRIEF-[YourName]-[Date].md — your dashboard needs

Forward both files to your supervisor (email, Slack, or however your team normally shares files). Do this right away so it doesn't slip.

✓
You're done when: Both .md files are saved and forwarded to your supervisor.
4

Start Your Grounding Package

~45 min

Step 4 of 4

The grounding package is where Claude gets to know you and your business. It has two parts: a profile interview and a dossier review.

Part A: Client Profile

Claude will have a conversation with you — not a form, an actual back-and-forth interview. It covers your business, responsibilities, tools, pain points, goals, and how you prefer to work with AI.

Open a Claude Code terminal panel and type:

RUN THIS COMMAND Run the client profile interview. Use the client-profile-interview skill. Save my profile to 00 System/CLIENT-PROFILE.md and save progress as we go.

Take your time. Be specific and honest — Claude uses your answers to build your entire AI configuration. If you need to step away, progress is saved automatically.

Part B: Your Dossier

After the profile interview, Claude creates a dossier — a summary of your setup, your business context, and what was accomplished. This gets sent to the team so they know exactly who you are before the program starts.

If the dossier wasn't generated automatically, run:

RUN THIS COMMAND Generate my client dossier. Create it at 00 System/CLIENT-DOSSIER.md with a summary of my profile, vault structure, documents, and setup status. Then show it to me for review before sending.

Review the dossier. When you're satisfied:

APPROVE & SEND The dossier looks good. Send it to the Strategic Profits team.
✓
You're done when: Claude confirms the dossier was sent. The team will review it before your program starts.
