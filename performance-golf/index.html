Five steps to get up and running. Start at the top and work your way down.

1 Book Your Call 2 Set Up Claude 3 AI Mapping 4 Client Profile 5 Dossier
1

Book Your Call with Ben

~5 min

Step 1 of 5

Your first step is a one-on-one call with Ben to map out your AI strategy and make sure you're set up for success.

Book your call at meetben.com/pg →

✓
You're done when: You have a confirmed call on the calendar.
2

Set Up Claude

~10 min

Step 2 of 5

Already have Claude Code working? Skip to Step 3.

You'll need Claude Code installed and connected to the SP Shared MCP server. Pick whichever method you prefer.

Option A: Desktop App

Download and install the Claude desktop app from claude.ai/download. Open it and sign in with your Anthropic account.

Option B: CLI (Terminal)

If you prefer working in a terminal, install Claude Code via npm:

npm install -g @anthropic-ai/claude-code

Then run claude in your terminal and follow the sign-in prompts.

Connect SP Shared MCP

Once Claude Code is running, paste this command, replacing YOUR_SP_API_KEY_HERE with your API key:

RUN THIS COMMAND Add this MCP server to my project config: name "sp-shared", URL "https://mcp.strategicprofits.com/mcp", with header "Authorization: Bearer YOUR_SP_API_KEY_HERE". Then run sp_sync in full mode to download shared skills.
✓
You're done when: Claude confirms sp_sync completed and skills were downloaded.
3

Complete Your AI Mapping

~20–30 min

Step 3 of 5

Before your call with Ben, complete the AI opportunity interview. This maps out your business, workflows, pain points, and where AI can make the biggest impact for you.

Open Claude Code and run:

RUN THIS COMMAND Run the AI opportunity interview. Use the ai-opportunity-interview skill. When the interview is complete, submit the results using the sp_submit_ai_opportunity tool.

Claude will have a 20–30 minute conversation with you — not a form, an actual back-and-forth interview. Answer honestly and specifically. There are no wrong answers. If you need to step away, progress is saved automatically.

When the interview is complete, Claude will automatically submit your results to the team.

✓
You're done when: Claude confirms your AI Opportunity Profile and Command Center Brief were submitted.
4

Client Profile

~20–30 min

Step 4 of 5

Now Claude gets to know you personally. This is a deeper interview covering your business, responsibilities, tools, pain points, goals, and how you prefer to work with AI.

Open Claude Code and run:

RUN THIS COMMAND Run the client profile interview. Use the client-profile-interview skill. Save my profile to CLIENT-PROFILE.md and save progress as we go. When complete, submit it using sp_submit_onboarding with program "pg".

Take your time. Be specific and honest — Claude uses your answers to build your entire AI configuration. If you need to step away, progress is saved automatically.

When the interview is complete, Claude will save your profile locally and submit it to the team.

✓
You're done when: Claude confirms your client profile was submitted.
5

Your Dossier

~10 min

Step 5 of 5

Claude creates a dossier — a summary of your setup, your business context, and what was accomplished. This gets sent to the team so they know exactly who you are before the program starts.

If the dossier wasn't generated automatically after your client profile, run:

RUN THIS COMMAND Generate my client dossier. Create it at CLIENT-DOSSIER.md with a summary of my profile, documents, and setup status. Then show it to me for review before sending.

Review the dossier. When you're satisfied:

APPROVE & SEND The dossier looks good. Submit it using sp_submit_onboarding with file type "dossier" and program "pg".
✓
You're done when: Claude confirms the dossier was submitted. The team will review it before your program starts.
